Detailed Description

Class implementation for running KMedoids methods.

KMedoids class. Creates a KMedoids object that can be used to find the medoids for a particular set of input data.

Parameters
nMedoidsNumber of medoids/clusters to create
algorithmAlgorithm used to find medoids; options are "BanditPAM" for the "Bandit-PAM" algorithm, or "naive" to use the naive method
verbosityVerbosity of the algorithm, 0 will have no log file emitted, 1 will emit a log file
maxIterThe maximum number of iterations the algorithm runs for
logFilenameThe name of the output log file

Build step for BanditPAM.

Runs build step for the BanditPAM algorithm. Draws batch sizes with replacement from reference set, and uses the estimated reward of the potential medoid solutions on the reference set to update the reward confidence intervals and accordingly narrow the solution set.

Parameters
dataTransposed input data to find the medoids of
medoid_indicesUninitialized array of medoids that is modified in place as medoids are identified
medoidsMatrix of possible medoids that is updated as the bandit learns which datapoints will be unlikely to be good candidates

Build step for the naive algorithm.

Runs build step for the naive PAM algorithm. Loops over all datapoint and checks its distance from every other datapoint in the dataset, then checks if the total cost is less than that of the medoid (if a medoid exists yet).
